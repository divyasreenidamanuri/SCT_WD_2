# ⏱ Stopwatch Web Application

A modern and fully responsive stopwatch web application developed as part of my **Web Development Internship at SkillCraft Technology**. This project focuses on creating an interactive and user-friendly stopwatch with accurate time tracking, lap recording, theme customization, and advanced functionality using HTML, CSS, and JavaScript.
